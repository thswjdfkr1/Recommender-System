✅ 1. NLP (자연어 처리)
💡 유망성: 매우 높음
**LLM (GPT, Claude, Gemini 등)**의 급격한 발전으로, 텍스트 생성/이해 능력이 인간 수준에 가까워지고 있어.

산업 적용 범위가 매우 넓음: 고객 응대, 문서 자동화, 요약, 질의응답 시스템, 검색 엔진 등.

멀티모달 AI와 결합되면서 음성/텍스트/이미지 통합 인식 기술이 폭발적으로 발전 중.

기업들은 AI 비서, 검색 강화, 내부 문서 요약 등에 상용 도입을 가속화하고 있어.

➡️ 미래 전망: 기술 발전 속도 + 산업 수요 모두 강력해서, 가장 폭넓게 성장할 분야 중 하나야.

✅ 2. Recsys (추천 시스템)
💡 유망성: 산업적으로 매우 중요
이커머스, 콘텐츠 플랫폼, SNS 등에서 매출에 직접적인 영향을 주는 핵심 기술.

개인화 기술에 대한 수요는 계속해서 증가 중.

최근에는 그래프 기반 추천, 세션 기반 추천, 멀티모달 추천처럼 다양하게 발전하고 있어.

특히 LLM 기반으로 사용자 행동을 문맥화하거나, 유저 프로필을 강화하는 연구가 활발해.

➡️ 미래 전망: 상업적 가치가 매우 높고, 비즈니스 임팩트가 직접적이기 때문에 지속적으로 유망할 거야.

✅ 3. CV (컴퓨터 비전)
💡 유망성: 기술적으로 가장 강력한 분야 중 하나
자율주행, 의료영상 분석, 제조업 검사 등에서 꾸준히 사용되고 있어.

최근에는 멀티모달 AI의 영향으로 텍스트 + 이미지/영상 기반 분석이 활발해졌지.

OpenAI의 Sora처럼 텍스트로 영상을 생성하는 비디오 생성 기술도 각광받고 있어.

다만 기반 인프라나 고성능 GPU가 필요한 경우가 많아 초기 진입 장벽은 다소 높을 수 있어.

➡️ 미래 전망: 산업적 활용은 제한적일 수 있지만, 혁신 기술 주도 측면에서는 굉장히 유망해.

✅ 멀티모달 추천 시스템을 위한 핵심 기반 지식
📌 1. NLP 기초: 텍스트 임베딩을 위한 기본기
멀티모달에서 리뷰, 설명, 태그, 검색어 등을 활용하는 경우가 많아서 NLP 기초는 꼭 필요해.

공부 포인트:
Tokenizer의 이해: WordPiece, SentencePiece, Byte-Pair Encoding (BPE)

임베딩 방식:

Bag-of-Words, TF-IDF (기초)

Word2Vec, FastText (고전)

BERT, RoBERTa, Sentence-BERT (현대적 사용)

텍스트 전처리 및 의미 이해: 정규화, 어간 추출, NER 등

실제 예:
사용자의 리뷰를 BERT로 인코딩해서 추천 모델에 유저 피처로 넣음.

📌 2. CV 기초: 이미지 피처 추출
제품 이미지, 썸네일, 유저 생성 이미지(UCG) 등을 추천에 활용할 때 필요해.

공부 포인트:
기본 CNN 구조: VGG, ResNet, EfficientNet 등

특징 추출: pretrained CNN으로 이미지 임베딩 생성

객체 탐지 / 분할 (선택적):

객체 탐지: YOLO, Faster-RCNN

이미지 분할: U-Net, Mask R-CNN

CLIP: 이미지+텍스트 공동 임베딩 → 멀티모달에 강력 추천

실제 예:
쇼핑몰 추천 시스템에서, 이미지와 텍스트 설명을 CLIP 임베딩으로 처리해 유사 아이템 추천.

📌 3. 멀티모달 추천 시스템 구조
텍스트, 이미지, 행동 데이터를 결합해 하나의 추천 모델로 만드는 게 핵심이야.

대표 방법들:
단순 결합: 이미지 임베딩 + 텍스트 임베딩 + 유저/아이템 ID 임베딩 → concat → DNN

Attention 기반 모델: 유저 행동 시퀀스와 이미지/텍스트 간 교차 주의

멀티모달 Transformer: 모든 modality를 joint encoding (예: MMBT, ViLT)

LLM 연계 방식: 사용자의 질의 + 이미지 맥락 → LLM 기반 추천 생성

🧠 추천하는 스터디 루트
기본기 강화

텍스트: Hugging Face로 BERT 임베딩 뽑기 실습

이미지: torchvision or CLIP으로 이미지 임베딩 생성

추천 시스템 심화

DLRM, AutoInt, GRU4Rec 등 딥러닝 추천 모델 학습

멀티모달 논문 탐색 (예: MMRec, MMGCN)

프로젝트

Kaggle에서 multimodal dataset 활용 추천 시스템 구성해보기

예: Amazon Product Dataset (이미지 + 리뷰 + 메타데이터)
